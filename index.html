Detect Language With Python

30 March 2013

Found langid.py library to identify language when I working on one fun project. This library can we use for detecting language from one or more sentences.

The usage is very easy, just do:

  >> import langid
  >> print langid.classify("Aku cinta semua cinta buatan Indonesia")
  ('id', 0.68797031656292873)

Isn't it cool?! I used this library to detect language from twitter stream.
